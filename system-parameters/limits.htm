LIMITS

You can set the limits for various parameters to prevent invalid data to be entered.
These ensures data correctness & consistency and eliminated errors in data
The system should not allow abnorbal data to be saved

Menu Location

The functionality is accessible from the limits sub-menu.
Settings> System parameters > Limits



The upper and lower limits are defined.
When a user inputs data on a field whose limits are being tracked, any breach in limit will result to a warning message and the use will be unable to commit the data


The limits can be adjusted on a need basis by clicking the open window icon.
This will display a screen where the user can edit the limits and save the new limit configurations.
Adjust appropriately then save.

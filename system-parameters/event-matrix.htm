EVENTS MATRIX

The event matrix dictates which event is application to an animal type. i.e. insemination & milking events for cows but not bulls.
A user will have the control of configuring the matrix.

Menu Location

Settings> System Parameters> Events Matrix

A value of 1 means availability while a value of 0 denotes un-availability.

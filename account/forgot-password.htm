FORGOT PASSWORD

If a system user is unable to successfully login into the application because of wrong credentials supplied, the system has a facility to enable the password to be reset
On the sign in page, at the bottom, there is a link to the forgot password page



Click on the link. This will open the password reset page

password-reset



The user will input email address. The system will check if an account with the email supplied exists.
If it doest exists, the system will warn the user that an accountwith the email supplied does not exists.



If the the account exists, the password will be reset and an email with the new passowrd will be sent to the user's email address.

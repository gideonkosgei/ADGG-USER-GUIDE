CHANGE PASSWORD

The system requires users to change their passwords once in a while
The password change screen is accessed by clicking the user profile avatar. This will display the account information screen.
Click on the security table the the page will be displayed




A strong password must have:

  1. Lowercase
  2. Uppercase
  3. Symbol or special character
  4. Number
  5. At least 10 characters in length

Steps

  1. Enter current password
  2. Enter new password
  3. Confirm new password

The save changes button will only be enabled if the new password & the confirm new password match
As you enter the new password, the password strength meter will be responsive to the input



After inputing new the password & confirming, save changes
If isystem will check if the current password entry matches with what is saved on the database.

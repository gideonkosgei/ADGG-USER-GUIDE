CALENDAR

The application has a system calendar that is used for :

  1. Scheduling
  2. Planning
  3. Act as a reminder

The calendar items can be added
  1. Automatically - Calendar items are created by a background process
  2. Manually - Involves a user creating the event explicitly.

Menu Location



Calendar Elements



Create Calendar Events

Click on the add event button and a modal window to capture the new event will pop.
Populate the details of the new event and click add to save the event. Go back to the calendar view and locate the view.


Edit/Delete Calendar Event

Double click on the event on the calendar body.
A modal window with the calendar item clicked will pop.
At the bottom of the form, there is a delete/cancel/save buttons.
Choose the appropriate action by clicking a button

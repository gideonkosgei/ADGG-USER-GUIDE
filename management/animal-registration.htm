view animal details & events menus

On the animal grid, there is an 'open new' icon (1) which when clicked, it displays the details on the animal selected.
.
.
The animal details window will display a form populated details of the animal selected.
From this window, a user can:
  1. Edit details
  2. View Record metadata
  3. Capture animal events
The events menu is displayed at the top of the form.
.
.
Below is a detailed description of the events menu.
.
.
The events menu icon visibility is control by events menu matrix that dictated which event applied to which category/type of animal.
For events menu matrix check the settings section.

Edit animal Details

Once a user has landed on the animal details screen, all the details of the animal will be displayed on a form.
By default, the form is disabled and does not accept any input.
Based on the user system rights & privileges, the form activated by toggling the enable/disable toggle switch found at the bottom of the screen.
.
.
A disabled form does not display the save details button. It is only visible once the form has been enabled.
.
.
Once the switch has been turned on, the form becomes active and ready to accept input.
Also, the save changes becomes visible. The user can modify the animal details as appropriate the save the changes.

View Record Metadata

You can view a record's metadata. The data captured include:
  1. Date created
  2. The record creator
  3. The last record update date
  4. The user who last updated the record
On the Record details screen, at the bottom, an icon to open the metadata modal is displayed.
.
.
On the click of the icon, a modal will pop that contains data that describes the record.
.

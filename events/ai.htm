INSEMINATION

Access the Insemination Event Screen by clicking the insemination icon on the animal detail toolbar.
A grid will be displayed that list previous insemination events for the animal.



Add New Record

To add a new insemination event, click on the add new icon on the insemination event listing grid toolbox.



Capture the details of the new Insemination event and save.
The new record will be listing on the insemination events grid.

MILKING

Access the Milking Event Screen by clicking the Milking icon on the animal detail toolbar.
A grid will be displayed that list previous Milking events for the animal.



Add New Record

To add a new Milking event, click on the add new icon on the Milking event listing grid toolbox.



Capture the details of the new Milking event and save.
The new record will be listed on the Milking events grid.
